# GerenciamentoDeIO
 Um desafio de manipulação de arquivos em java onde o intuito era: 
 
 1 - Conseguir saber quantas vezes ocorria uma leitura/escrita de arquivo; e
 2 - Conseguir capturar o tempo de cada operação.
